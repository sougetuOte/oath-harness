# フォーカスモード

現在のタスクに集中するため、必要最小限の情報のみを表示する。

## 実行ステップ

1. **現在のタスクを特定**
   - 進行中のタスクを1つに絞り込む
   - そのタスクに必要な技術情報のみを特定
   - 不要な背景情報はロードしない

2. **集中作業の提案**
   - 特定したタスクの具体的な次のステップを提案
   - 25分のポモドーロタイマーを推奨
   - 他のタスクには言及しない

3. **簡潔な回答**
   - 回答は3〜5文以内
   - 具体的なアクションのみを提示
   - 長い説明や背景情報は省略

## 優先順位
1. 緊急かつ重要（即座に対応）
2. 重要だが緊急でない（計画的に対応）
3. その他は一時保留

## 回答例
```
現在のフォーカス: [タスク名]
次のアクション: [具体的な作業]
所要時間目安: [分]
完了条件: [明確な基準]
```

詳細情報が必要な場合は、明示的に「詳細情報が必要」とリクエストすること。
