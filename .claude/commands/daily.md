# デイリーアップデート

3分以内で日次の軽量ステータス更新を完了する。

## 実行ステップ

1. **現状更新**
   - 本日の進捗を確認
   - 完了したタスクを記録
   - 新しい学びがあれば記録

2. **明日の準備**
   - 明日の最優先タスクを1つ決定
   - ブロッカーがあれば記録

3. **簡潔な振り返り**
   - 本日完了したタスク（1〜2項目）
   - 明日の最優先タスク（1項目）
   - 課題があれば（最大1つ）

## テンプレート
```
## [日付] デイリーアップデート

### 本日完了
- [タスク1]
- [タスク2]

### 明日の最優先
- [タスク名] - [理由]

### 課題・気づき
- [あれば最大1つ]

### 作業時間
- 実働時間: [時間]
- 効率: 高/中/低
```

## 更新不要な情報
- プロジェクト概要
- 技術スタック
- 長期計画
- 詳細な技術情報

3分以内に完了できない場合は、情報を絞り込むこと。
重要でない情報の更新は週次でまとめて行う。
